## 카카오톡 예약 메시지 설정

1. **데이터베이스 설정**: Supabase 대시보드에서 `supabase-schema.sql` 파일의 내용을 실행하여 테이블을 생성합니다.

2. **카카오톡 API 연동**: `KAKAO_API_SETUP.md` 파일을 참고하여 백엔드 서버를 구축하고 카카오톡 API를 연동합니다.

3. **사용 방법**:
   - 받는 사람, 메시지 내용, 예약 날짜/시간을 입력합니다.
   - "예약 메시지 등록" 버튼을 클릭하여 예약합니다.
   - 예약된 메시지는 목록에서 확인할 수 있으며, 삭제할 수 있습니다.
   - 지정된 시간에 자동으로 메시지가 전송됩니다 (백엔드 스케줄러 필요).
